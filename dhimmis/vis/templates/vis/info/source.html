This view shows the sources from which the evidence tuples were extracted. An evidence tuple can stem from more than one source, and therefore the total sum of all source instances can be larger than the number of evidence tuples.

Contents

The view consists of a table in which each row represents one source. The rows are sorted by descending number of evidence tuples.

Checkboxes

The first column contains a checkbox that toggles the visibility of evidence tuples of a source. Keep in mind that an evidence tuple can have multiple source instances. As long as any source instance's source is checked in this view, the evidence tuple will be active.

Short Names

The second column shows the short name of the source, generally an abbreviation. The full name of the source is available via a tooltip.

Religion/Confidence Segmentation Visualization

The third column visualizes the segmentation of the evidence tuples on the religions or on the levels of confidence, depending on the mode. In Confidence Mode, the currently selected aspect of confidence is used. The visualization used is a normalized stacked barchart. The width of each segment signifies the number of evidence tuples with that religion or confidence level in relation to the total evidence count for that source. The total width of the bars is constant regardless of the amount of evidence tuples.

Evidence Count Visualization

The fourth column visualizes the count of evidence tuples for each source. This column uses the same scaling in all rows, effectively creating a vertical histogram across the rows.

Interaction

Selection

Clicking on a row will select the source, and link related data in all views. For instance, locations in the map with evidence from that source will be highlighted. Clicking on the same row again will reset the selection. Selecting a different row, or an element in a different view, will replace the selection.

Filtering

Sources can be filtered out from the visualized data in all views by unticking the appropriate checkbox or checkboxes. The selection of checked sources is not applied until the Apply button is clicked. Checking multiple (or all) sources works as a logical disjunction, that is, an evidence tuple is matched if it stems from any of the checked sources. The source filters work together with filters from other views in logical conjunction; for instance, if only source X and religion Y are selected in the respective views, only evidence tuples with that religion and source are matched.

Three additional buttons in the top right make filtering more convenient:

Uncheck all
Uncheck all checkboxes.
Invert
Invert all checkboxes. Checked boxes are unchecked, unchecked boxes are checked.
Check all
Check all checkboxes.
