This view shows the sources from which the evidence tuples (i.e. a set of data concerning a place, religion, time span, and the respective confidences) were gathered. Note that an evidence tuple can stem from more than one source. Therefore, the sum total of all source instances can be larger than the number of evidence tuples.

Contents

The view consists of a table in which each row represents one source. The rows are sorted by descending number of evidence tuples. In what follows, the respective columns of one given row are described.

Checkboxes

The first column contains a checkbox that toggles the visibility of evidence tuples of a source. Keep in mind that an evidence tuple can stem from more than one source: as long as any source associated with the evidence tuple is checked in this view, it will be active.

Short Names

The second column shows the short name of the source, generally an abbreviation. The full name of the source with bibliographical details is available via a tooltip, i.e. when hovering over the name with the mouse.

Religion/Confidence Segmentation Visualization

The third column visualizes the segmentation of the evidence tuples on the religions or on the levels of confidence, depending on the mode (cf. "Display mode" in the Settings pane. In Confidence Mode, instead of religion, the currently selected aspect of confidence is used.) The visualization used is a normalized stacked barchart. Because the total width of the bars is the same regardless of the amount of evidence tuples, the width of each segment signifies the number of evidence tuples with that religion or confidence level in relation to the total evidence count for that source.

Evidence Count Visualization

The fourth column visualizes the count of evidence tuples for each source. This column uses the same scaling in all rows: the longer the bar, the more evidences stem from the respective source. This effectively creates a vertical histogram across the rows.

Interaction

Selection

Clicking on a row will select the source and link related data in all views. For instance, locations in the map with evidences from the selected source will be highlighted, other locations will be represented by less saturated colors. Clicking on the same row again will reset the selection. Selecting a different row, or an element in a different view, will replace the selection. Only one source can be selected at a time.

Filtering

Sources can be filtered out from the visualized data in all views by (un-)checking the appropriate checkbox or checkboxes. Note that the selection is not applied until the Apply button is clicked. Checking multiple (or all) sources works as a logical disjunction: an evidence tuple is matched if it stems from any of the checked sources. The source filter works together with filters from other views in logical conjunction; for instance, if only source X and religion Y are selected in the respective views, only evidence tuples with source X and religion Y are matched.

Three additional buttons in the top left corner make filtering more convenient:

Uncheck all
Uncheck all checkboxes.
Invert
Invert all checkboxes. Checked boxes are unchecked, unchecked boxes are checked.
Check all
Check all checkboxes.
