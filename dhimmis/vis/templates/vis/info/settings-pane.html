Visualization Settings

These are settings that directly affect the visualization. In particular, they control which data is shown, and in what way:

Show filtered data

This controls if data which does not match the currently selected filters is also visualized. If All data is selected, non-matched data is represented by less saturated colors.

Display mode

This controls which aspect of the data is mapped to color. If Religion is selected, the religion is used for coloring, using the color scheme in the Religion tab. If Confidence is selected, the level of confidence is used, using the color scheme in the Confidence tab. The aspect of confidence used can be selected in the Confidence tab as well, and the currently used aspect is indicated with an eye symbol below the column.

Timeline mode

This switch controls whether the timeline shows a qualitative summary or quantitative information. If Quantitative is selected, the number of evidences of that type in that year is encoded into the height of the area. The timeline then looks like a stacked area chart. If Qualitative is selected, the timeline only shows whether there are evidences of that type each year. The timeline then looks like a Gantt chart.

Map mode

This switch controls whether the map shows large, detailed symbols for clusters of location without overlap, or one smaller symbol per place. The latter mode may lead to overlap of symbols and should be used at one's own peril.

Generate Report

You can generate a detailed text report of all the data visible under the currently selected filters. The report will also contain information on the filter criteria that lead to the selection of data. This report will show up in a new tab. You can also create a report by uploading a visualization state (see below) you have exported. Be careful not to do this with very broad filter criteria, as the report will get very long in that case.

This section also has a button labeled Describe filters. Clicking this button will open a modal window within the visualization that describes the currently active filters in text form. This description is the same as at the top of a generated report.

Layout Settings

You can save the current layout. This setting will only persist for your current browser and computer (using localStorage). You can also reset to the initial layout (Note: this will reload the page).

Persist State

You can also save the current state of the visualization. In particular, this will save the filters you currently have applied, the center point, zoom level, and visible layers of the map, and the visualization settings applied above. These settings are downloaded as a file to your computer. You can upload such a file later to restore the state. You can also submit that file to generate a report based on the filters later on.
