In this view, different settings and features can be controlled.

Visualization Settings

These are settings that directly affect the visualization. In particular, they control which data is shown, and in what way.

Show filtered data

This switch controls if data not matching the currently selected filters is also visualized. If All data is selected, non-matched data is represented by less saturated colors.

Display mode

This switch controls which aspect of the data is mapped to color. If Religion is selected, religion is used for coloring, using the color scheme in the Religion view. If Confidence is selected, the level of confidence is used, using the color scheme in the Confidence view. The Aspect of Confidence used can be selected in the Confidence view as well; there, the currently used aspect is indicated with an eye symbol below the column.

Timeline mode

This switch controls whether the timeline shows a qualitative summary or quantitative information. If Quantitative is selected, the number of evidences of that type in that year is represented by the height of the area. The timeline then looks like a stacked area chart. If Qualitative is selected, the timeline only shows whether there are evidences of that type each year. The timeline then looks like a Gantt chart.

Map mode

This switch controls whether symbols are clustered or unclustered. By default, symbols are clustered: zooming out will lead to religions attributed to different places being summarized according to ###. If symbols are unclustered, one small symbol per place and religion is shown. This mode may lead to overlap of symbols when zooming out even to a relative small scale and should thus be used with due caution when interpreting the results.

Generate Report

With this feature, a detailed text report of all the data visible according to the currently selected filters can be generated. The report will also contain information on the filter criteria that led to the selection of data. This report will show up in a new tab. A report can also be created by uploading a visualization state that was safed and downloaded beforehand (see below). Be careful not to do this with very broad filter criteria, as the report will get very long in that case.

This section also has a button labeled Describe filters. Clicking this button will open a modal window within the visualization that describes the currently active filters in text form. This description is the same as at the top of a generated report.

Layout Settings

You can save the current layout. This setting will only persist for your current browser and computer (using localStorage). You can also reset to the initial layout (Note: this will reload the page).

Persist State

Also, the current state of the visualization can be saved. In particular, this will save the filters you currently have applied, the center point, zoom level, and visible layers of the map, and the visualization settings applied above. These settings are downloaded as a file to the computer. Later, such a file can be uploaded to restore the state.
